# Round Integral (RealityKit)

Rounds X to integral value using round ties to even rounding mode in floating-point format.

iOS 17.0+iPadOS 17.0+Mac Catalyst 17.0+macOS 14.0+tvOS 26.0+visionOS 1.0+
